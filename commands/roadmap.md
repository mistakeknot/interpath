# Generate Roadmap

Invoke the `artifact-gen` skill with artifact type **roadmap**.

Before generating the roadmap, run the sync script to refresh `docs/roadmap.json`:

```bash
ROADMAP_SYNC="${CLAUDE_PLUGIN_ROOT}/scripts/sync-roadmap-json.sh"
if [ -z "${CLAUDE_PLUGIN_ROOT:-}" ] || [ ! -x "$ROADMAP_SYNC" ]; then
    echo "Warning: CLAUDE_PLUGIN_ROOT not set or sync script not found, skipping JSON sync" >&2
else
    bash "$ROADMAP_SYNC"
fi
```

The roadmap synthesizes:
- Current project state (version, components, companions)
- Open beads organized by priority into Now/Next/Later phases
- Research agenda from brainstorms and flux-drive summaries
- Blocking relationships and dependency graph

**Monorepo mode:** When run from a monorepo root, this command auto-detects the monorepo context and generates an ecosystem-wide roadmap covering all discovered sub-modules. Use `/interpath:propagate` to push items back to individual module roadmaps.

Output:
- `docs/interverse-roadmap.md` (human-facing canonical view for monorepo root)
- `docs/roadmap.json` (machine-readable canonical source, for downstream tooling)
- `docs/roadmap.json` is generated by `${CLAUDE_PLUGIN_ROOT}/scripts/sync-roadmap-json.sh` and should be treated as the canonical feed.

Use the Skill tool to invoke `interpath:artifact-gen` with the instruction: "Generate a roadmap artifact."
